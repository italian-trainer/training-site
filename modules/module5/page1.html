Host Sequence of Service (SOS)

  1. Greeting the Guests
  2. Walking to the Table
  3. Delivering the Menus
  4. Alerting the Server
  5. Thank the Guest

1. Greeting the Guests

You get the first opportunity to make our guests feel welcome as they come through the door. You’re their first impression so it needs to be a good one!

  • Every guest should get a warm, smiling welcome.
    “Hi, welcome to Maria’s! How are you doing this evening?”
  • You also need to identify how many guests are in the group. Sometimes it seems obvious but you never want to assume, so it’s always good to ask.
    “Two for dinner this evening?”
  • Once you know how many guests, you can select a table where you would like to seat them.
    You should already have a table in mind that fits with your rotation. Always be thinking ahead, so that when the guest arrives you can get them to a table quickly.
    Make sure that the table is completely prepared and ready for guest seating before you start toward it with guests. Is it fully set for the right number? Is it ready with any needed high chairs or booster seats?
  • Now you just need to grab the appropriate number of menus and lead the guests through the dining room to the table.
    “Right this way.”
    OR
    “Just follow me and we’ll get you to a table.”

Occasionally, guests may seat themselves, bypassing the host stand and heading straight for the table they want. If you’re in a restaurant with multiple entrances, this might be more common, so it’s important for you to be aware of guests entering through both doors.

It’s ok, when that happens. You just need to follow them to the table with the menus they’ll need and welcome them to Maria’s just like you would any other guest. It’s also more likely in these instances that there might be a break in your planned seating rotation or that they’ve selected a table that isn’t currently assigned to a server. If that’s the case, make sure you adjust your rotation accordingly, and make sure the table gets picked-up by a server.
